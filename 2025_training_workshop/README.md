# 2025 WheatCAP UAS Training Workshop

> Instructor: Jinha Jung, Anjin Chang, Ismaila Olaniyi, Jean-Luc Jannink

## Training Schedule

1. Introduction to T3
2. Introduction to D2S
   1. D2S account creation
   2. Create a demo project
   3. Raw data processing
   4. Plot boundary delineation using QGIS
   5. Pull T3 field layout information
   6. Merge *plot_id* from T3 field layout into the plot boundary
   7. Upload the merged plot boundary to the D2S project
   8. Extract phenotypic traits on D2S
   9. Perform zonal statistics
   10. Download the zonal statistics
   11. Push the zonal statistics back to T3
   12. Link D2S project with a T3 trial

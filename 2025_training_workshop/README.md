# 2025 WheatCAP UAS Training Workshop

> Instructor: Jinha Jung, Anjin Chang, Qiusheng Wu, Ismaila Olaniyi, Jean-Luc Jannink

## Training Schedule

1. Introduction to T3
2. Introduction to D2S
   1. D2S account creation
   2. Create a demo project
      * You can download sample raw data from the *2025 WheatCAP Training Workshop* project (https://ps2.d2s.org/projects/900c7d94-1b7b-460e-9a85-19359736c445/flights/e37199a5-1e8a-460b-91c9-56dcaf97f73f/data)
   4. Raw data processing
   5. Plot boundary delineation using QGIS
   6. Pull T3 field layout information
   7. Merge *plot_id* from T3 field layout into the plot boundary
   8. Upload the merged plot boundary to the D2S project
   9. Extract phenotypic traits on D2S
   10. Perform zonal statistics
   11. Download the zonal statistics
   12. Push the zonal statistics back to T3
   13. Link D2S project with a T3 trial
